# 🤫 The Quiet Hustle

Faceless affiliate marketing education brand. All design assets, content templates, and brand materials.

## Brand Identity
- **Voice:** Calm authority. Insider knowledge. Zero hype.
- **Aesthetic:** Luxury editorial minimalism. Dark-first. Gold accent.
- **Platforms:** Instagram (primary), TikTok (cross-post), Pinterest (evergreen)

## Design Tool
All visual assets are designed in **Claude** (claude.ai Projects), not Canva.

## Workflow
1. Script content in Notion (Content Calendar DB)
2. Generate graphics in Claude Project (using brand kit instructions)
3. Export and save to `/content/week-XX/` and `/exports/`
4. Edit Reels in CapCut (overlay frames from `/templates/reel-overlay/`)
5. Schedule via Meta Business Suite
6. Track status in Notion

## File Naming
`tqh-{format}-{topic-slug}-{slide-number}.{ext}`

Examples:
- `tqh-carousel-affiliate-red-flags-01.png`
- `tqh-reel-passive-income-myth.png`
- `tqh-static-first-100-dollars.png`
